Points Breakdown by Position

🧤

Goalkeeper

Clean sheet: +6pts. Save: +1pt each. Penalty save: +5pts. Goal conceded (per 2): -1pt. Yellow card: -1pt. Red card: -3pts.

🛡️

Defender

Clean sheet: +4pts. Goal scored: +6pts. Assist: +3pts. 60+ mins: +2pts. Yellow card: -1pt. Red card: -3pts.

⚡

Midfielder

Goal scored: +5pts. Assist: +3pts. Clean sheet: +1pt. 60+ mins: +2pts. Yellow card: -1pt. Red card: -3pts.

🎯

Forward

Goal scored: +4pts. Assist: +3pts. 60+ mins: +2pts. Missed penalty: -2pts. Yellow card: -1pt. Red card: -3pts.
